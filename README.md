# the-power-of-music

React(Frontend) + Node.js(Backend)(Express js)

## Development
### Backend
```bash
cd the-power-of-music-backend
yarn
mkdir uploads
yarn start:dev
```
### Frontend
```bash
cd the-power-of-music-frontend
yarn
yarn start
```
  
## Production
### Frontend
```bash
cd the-power-of-music-frontend
yarn
yarn build
```

### Backend
```bash
cd the-power-of-music-backend
mkdir uploads
yarn deploy
```
